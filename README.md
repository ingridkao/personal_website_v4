# personal_website_v4

Look at the [Content documentation](https://content-v2.nuxtjs.org/) to learn more.

## Setup
```bash
# npm
npm install
```

## Development Server
```bash
npm run dev
```
Start the development server on http://localhost:3000

## Production
Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
